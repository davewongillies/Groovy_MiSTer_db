# Groovy\_MiSTer\_db

Install Groovy\_MiSTer with `update_all`

## Setup

Put the following in `/media/fat/downloader.ini`:

```ini
[davewongillies/Groovy_MiSTer_db]
db_url = https://raw.githubusercontent.com/davewongillies/Groovy_MiSTer_db/db/db.json.zip
```

Then add this to `/media/fat/MiSTer.ini`

```ini
[Groovy]
main=MiSTer_groovy
```
